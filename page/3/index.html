词法作用域和动态作用域的详解

发表于 2012-09-11   |   分类于 javascript   |  

作用域有两种常见的模型，一种叫做词法作用域 Lexical Scope，一种叫做动态作用域Dynamic Scope。其中词法作用域更常见，被大多数语言采用，包括javascript。

词法作用域

词法分析过程Lex-time，是指系统讲源码字符串解读成有含义的token的过程。词法作用域就是说在词法分析过程(即代码解析时)中指派的作用域，词法作用域在词法解析过程中就已经定死了。
